# OSSCAP2020

1. 파일 실행에 필요한 python3 모듈 설치 
 - pip3 install pygame
 - pip3 install turtle

2. 게임 실행
 - python3 game_ver6.py
 
3. 조작 방법
  - T-Rex Run
    * 방향키 'UP'   : FireBall 발사(아이템을 획득한 경우에만)
    * 방향키 'DOWN' : 엎드리기
    * SPACE BAR     : JUMP
  - Catch Mind
    * 마우스 이용하여 오른쪽 색깔 리스트에서 원하는 색상 선택
    * 'z'를 누르면 펜이 지우개로 변경
    * 'SPACE BAR'를 누르면 그림 그리기 종료(이후에는 수정 불가)
